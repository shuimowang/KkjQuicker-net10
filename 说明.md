# KkjQuicker-net10

KkjQuicker 核心类库，基于 .NET 10.0 + WPF，为 Quicker 桌面动作和相关桌面应用提供基础能力封装。

## 目标框架

- `net10.0-windows`
- WPF + Windows Forms
- x64

## 构建

```bash
dotnet build KkjQuicker.csproj
```

要求 .NET 10 SDK。

## Quicker 依赖

本项目专门配合 Quicker 开发，项目文件中指向 `C:\Program Files\Quicker\*.dll` 的本机绝对引用是刻意设计，不是待修复问题。

这些引用使用 `<Private>false</Private>`，表示运行时由 Quicker 安装目录或宿主环境提供对应 DLL；除非项目目标从 Quicker 专用类库改成独立分发类库，否则不要把这类引用改成 NuGet 或本地复制输出。

## 目录职责

- `AI`：OpenAI-compatible Chat、实时 ASR 等 AI 能力。
- `Audio`：麦克风、系统回环等音频采集能力，只输出音频帧，不绑定业务。
- `Domain`：跨模块共享的轻量应用状态和领域模型。
- `Net`：HTTP、SSE、WebSocket 等网络能力。
- `Overlay`：覆盖层、截图、窗口贴边、侧边栏宿主等桌面交互能力。
- `UI`：WPF 控件、行为、转换器、Adorner、热键和 UI helper；不要放业务逻辑、数据访问或网络请求。
- `Utilities`：文件、剪贴板、Win32、Hook、图像、FFmpeg、线程限流等通用工具。
